Class FiboHeap

object --+
         |
        FiboHeap


This is an implementation of a Fibonacci Heap. A Fibonacci Heap is
a very efficient heap. The cost of an insert is O(1), and the amortized
cost of an extract minimum is O(lgn). You can extract an already inserted
item out of order in O(lgn). The way the Fibonacci heap obtains this is
by delaying the organizing of the items until you extract.

This code was modified and than ported to python with kind permission from:
- John-Mark Gurney (http://resnet.uoregon.edu/~gurney_j/jmpc/fib.html)

Extract the Min Element from the Fibonacci Heap. Functions returns the heap min element structure, and deletes it from the heap.

Get the Min Key on the Fibonacci Heap. Function returns the integer key of the data at the top of the Fibonacci Heap without extracting it.

Insert value into Fibonacci Heap. Function creates the data element based on provided values and insers into the heap. The pointer returned can be used in calls to functions like delete and decrease key.
